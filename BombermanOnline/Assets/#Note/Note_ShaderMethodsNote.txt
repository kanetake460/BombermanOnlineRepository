Shaderに使う関数をメモします。


〇絶対値
「abs()」
・絶対値を返す


〇小数切り捨て
「floor()」
・小数第一を切り捨てた値を求めます


〇小数部分
「frac()」
・小数部分を返します


〇内積（ドット積）
「dot()」
・スカラーのドット積を返します
【例】
// 法線ベクトルと、カメラのベクトルの内積を求める
dot(IN.viewDir, IN.worldNormal);


〇浮動小数点の剰余
「fmod()」
・floatの数値を割ったあまり


〇距離
「distance()」
・距離を求める
【例】
// ワールドの中心との距離を求める
distance(fixed3(0,0,0), IN.worldPos);


〇サイン、コサイン
「sin()」「cos()」
・サイン関数、コサイン関数は、入力値に対して周期的な波形を生成し、出力値を-1から1の範囲にマップします
・距離や時間に基づく周期的なパターンを生成することができます。
【例】
// 距離と時間を使って計算した値を-1～1で表す
float val = sin(dist * 3.0 - _Time * 100);
